# To do app in Vue.js

Simple to do list made with Vue and Vuetify following https://www.youtube.com/watch?v=CjXgoYo86yY

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
